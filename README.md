# AITuber Radio v0.8.0

**Chrome Extension だけで AITuber ラジオを配信・収録・自動生成できるツール。**

VRM / PNGTuber キャラクター × VOICEVOX / SBV2 / ブラウザ内蔵TTS × LLM AI フリートークで、  
台本再生・朗読・AI雑談・コメント読み上げ・**自動番組生成**をすべてローカルで実現します。

**[📻 Radio Station](https://k3peta.github.io/aituber-radio-station/)** — Web から番組をワンクリック再生

## 🚀 クイックスタート

### 必要なもの

| ソフト | 用途 | 必須 |
|---|---|---|
| Google Chrome | Extension 実行環境 | ✅ |
| [VOICEVOX](https://voicevox.hiroshiba.jp/) or [SBV2](https://github.com/litagin02/Style-Bert-VITS2) | 音声合成 (TTS) | △（ブラウザTTSも可） |
| OBS Studio | 配信 | 配信時のみ |
| [わんコメ](https://onecomme.com/) | コメント取得 | コメント機能使う時 |
| LLM API キー | AI フリートーク・自動番組生成 | AI 機能使う時 |

> **VOICEVOX なしでも使えます！** ブラウザ内蔵TTS（Web Speech API）を選択すれば、インストール不要で音声再生できます。Chromebook にも対応。

### インストール手順

1. Chrome で `chrome://extensions` を開く
2. 「**デベロッパーモード**」を ON
3. 「**パッケージ化されていない拡張機能を読み込む**」→ このフォルダを選択
4. ツールバーの 📻 アイコンをクリック → ポップアップが開く

## 📖 基本的な使い方

### Step 1: ビューワーを起動

ポップアップの「🖥️ ビューワーを開く」をクリック。  
VRM キャラクターが表示されるタブが開きます。

### Step 2: コンテンツを読み込む

#### A. プロジェクトフォルダごと読み込む（推奨）

1. ポップアップの「📂 フォルダ」をクリック
2. プロジェクトフォルダを選択
3. フォルダ内の台本（`.md`）、朗読台本（`readings/`）、メディア（`media/`）を一括認識
4. 台本が複数ある場合は**プルダウン**で選択可能

#### B. 台本ファイルを直接読み込む

1. ポップアップの「📖 台本 (.md)」をクリック
2. セットリストまたは台本の `.md` ファイルを直接選択

#### C. Radio Station から再生

1. [Radio Station](https://k3peta.github.io/aituber-radio-station/) にアクセス
2. 拡張機能IDを接続（ポップアップの📋コピー → Station に貼り付け）
3. 番組カードの ▶ ボタンをクリック

### Step 3: 再生する

- **▶️ 再生** — 選択中の台本を再生
- **⏹️ 停止** — 再生停止
- **⏱️ 開始ウェイト** — 0〜60秒のカウントダウン後に再生開始（OBS切替用）

## 🤖 自動番組生成（おはようラジオ）

LLM と各種 API を使って、**ボタンひとつで番組を自動生成・再生・録画**できます。

### 機能

- 📰 **ニュース自動取得**（Google News RSS）
- 🌤️ **全国天気予報**（気象庁データ — 北海道〜沖縄 10地域）
- 📅 **今日は何の日**（記念日・誕生花・偉人誕生日）
- 🤖 **LLM台本生成**（設定済みのAIが15分の台本を自動作成）
- 📹 **自動録画**（ダイアログなしでタブキャプチャ → WebM保存）
- ⏰ **毎日スケジュール実行**（指定時刻に自動生成）

### 使い方

1. ポップアップの **🤖 自動化** セクションを開く
2. **🌅 おはようラジオ生成 & 再生** をクリック
3. （オプション）**☑ 録画も行う** にチェック → WebMファイルが自動ダウンロード
4. （オプション）**☑ 毎朝自動実行** + 時刻設定 → 毎日自動で番組生成

### 番組構成

時間帯に応じて番組名が自動で変わります：

| 時間帯 | 番組名 | 挨拶 |
|--------|--------|------|
| 5:00〜11:59 | おはようラジオ | おはようございます |
| 12:00〜17:59 | ひるまちラジオ | こんにちは |
| 18:00〜4:59 | 夜ふかしラジオ | こんばんは |

### 番組の流れ

1. 挨拶 + 日付紹介
2. 今日は何の日コーナー（記念日・誕生花・偉人誕生日）
3. ニュースコーナー（2〜3本、AI の感想付き）
4. 全国天気予報（北海道から沖縄まで）
5. 今日の一言
6. 締めの挨拶

## 🖼️ PNGTuber

VRM モデルの代わりに、PNG 画像でキャラクターを表示する PNGTuber モードに対応しています。

### 使い方

ビューワーに **PNG 画像をドラッグ&ドロップ** するだけ。

| ドロップ | 結果 |
|---------|------|
| 画像 **2枚** | idle + talk で PNGTuber モード |
| 画像 **3枚** | idle + talk + blink（瞬き付き） |
| **Shift+ドロップ** 1枚 | PNGTuber モード（口パクなし） |
| 画像 1枚 | 背景変更 |
| **.vrm** ファイル | VRM モードに戻る |

### ファイル名の命名規則

自動判別するため、ファイル名にキーワードを含めてください。

| キーワード | 役割 |
|-----------|------|
| `talk`, `open`, `speak` | 口を開けた画像 |
| `blink`, `close` | 目を閉じた画像（瞬き） |
| 上記以外 | 通常（idle）画像 |

### クロマキー処理

グリーンバック画像は読み込み時に自動で透過処理されます。

### サンプル画像

`viewer/pngtuber/` に怪談ちゃんのサンプル画像（3枚）が同梱されています。

## 📹 録画機能

OBS なしで番組を動画として録画できます。

### 手動録画

1. ビューワー右上の「📹 録画」ボタンをクリック
2. Chrome の「**このタブを共有**」ダイアログが表示される → 「共有」をクリック
3. 録画開始（ボタン・ステータスバー・カーソルは自動で非表示）
4. 番組を再生
5. **放送終了時に自動で録画停止 → WebM ファイルが自動ダウンロード**

### 自動録画（tabCapture）

自動番組生成の「☑ 録画も行う」にチェックを入れると、**ダイアログなし**で自動録画されます。  
`chrome.tabCapture` API を使用し、ユーザー操作不要で完全自動化。

### 録画 Tips

- **開始ディレイを活用**: ポップアップで「⏱️ 5秒後に開始」を設定すれば、録画開始後にカウントダウン中はクリーンな待機画面が録れる
- **縦動画モード**: 台本ヘッダーに `size: 1080x1920` を指定すると自動でウィンドウリサイズ
- 形式: WebM（VP9 + Opus）、4 Mbps
- **YouTube にはそのまま投稿可能**（WebM対応）

## 🔄 WebM → MP4 変換ツール

X (Twitter) や TikTok 用に MP4 が必要な場合、同梱の変換ツールが使えます。

### GUIツール（macOS）

デスクトップの「**WebM→MP4**」アプリに WebM ファイルをドラッグ&ドロップ。  
円形プログレスバー付きで変換状況が確認できます。

```bash
# アプリを再生成する場合
python3 tools/webm2mp4_gui.py
```

### CLIツール

```bash
bash tools/webm2mp4.sh *.webm
```

- **VFR→CFR 変換**: 30fps 固定フレームレートで音ズレを防止
- コーデック: H.264 (libx264) + AAC
- `+faststart` でストリーミング最適化

> 要件: ffmpeg がインストールされていること（`brew install ffmpeg`）

## 📻 セットリスト（台本）の書き方

> 詳細仕様は [SETLIST_FORMAT.md](https://github.com/k3peta/aituber-radio-station/blob/main/SETLIST_FORMAT.md) を参照

### ヘッダー（YAML frontmatter）

```markdown
---
title: 番組タイトル
speaker: 38          # VOICEVOX スピーカー ID（38 = ずんだもん ヒソヒソ）
speed: 0.90          # 読み上げ速度
loop: false          # ループ再生
pause_between: 1500  # セグメント間の間隔 (ms)
size: 1080x1920      # 縦動画モード（オプション）
---
```

### セグメントタイプ一覧

| タイプ | 説明 | AI |
|--------|------|-----|
| `talk` | 台本トーク | 不要 |
| `jingle` | ジングル・SE 再生 | 不要 |
| `script` | 朗読 | 不要 |
| `freetalk` | AI フリートーク | 必要 |
| `reaction` | 朗読感想 | 必要 |
| `comments` | コメント読み上げ | 必要 |

### セグメント例

```markdown
# オープニング
[type: talk]
[bgm: media/bgm_chill.mp3]
[speaker: 3]
[emotion: happy]
[intensity: 0.7]
こんばんは！今日も楽しくやっていくよ！
```

### セグメント共通プロパティ

| プロパティ | 説明 |
|-----------|------|
| `speaker` | スピーカーID（セグメント単位で変更可） |
| `bgm` | BGM ファイル |
| `bgmVolume` | BGM 音量 (0.0〜1.0) |
| `bgmstop` | BGM 明示停止 |
| `emotion` | 表情（neutral/happy/sad/surprised/relaxed） |
| `intensity` | 表情の強さ (0.0〜1.0) |
| `overlay` | オーバーレイ画像 |

## 🔊 TTS エンジン設定

ポップアップの「🔊 TTS エンジン設定」から切り替えできます。

| エンジン | ポート | 備考 |
|---------|--------|------|
| **VOICEVOX** | 50021 | 高品質、要インストール |
| **Style-Bert-VITS2** | 5000 | カスタム音声モデル |
| **ブラウザ内蔵TTS** | — | インストール不要、Chromebook対応 |
| **カスタム** | 任意 | VOICEVOX 互換エンジン |

> ブラウザ内蔵TTS使用時の注意:
> - 口パクは音量シミュレーション（実音声解析ではない）
> - 録画時は「タブの音声も共有」にチェックが必要（手動録画の場合）
> - 自動録画（tabCapture）では自動的に音声取り込み

### 主なスピーカーID

| ID | キャラクター |
|----|------------|
| 2 | 四国めたん（ノーマル） |
| 3 | ずんだもん（ノーマル） |
| 8 | 春日部つむぎ |
| 29 | No.7（ノーマル） |
| 38 | ずんだもん（ヒソヒソ）← 通称「怪談ぺたん」 |

> 全スピーカー: VOICEVOX 起動後 `http://localhost:50021/speakers` で確認
