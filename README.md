**Chrome Extension だけで AITuber ラジオを配信・収録できるツール。**

台本再生・朗読・AI雑談・コメント読み上げをすべてローカルで実現します。

**[📻 Radio Station](https://k3peta.github.io/aituber-radio-station/)** — Web から番組をワンクリック再生

## 🚀 クイックスタート

### 必要なもの

| ソフト | 用途 | 必須 |
|---|---|---|
| Google Chrome | Extension 実行環境 | ✅ |
| [VOICEVOX](https://voicevox.hiroshiba.jp/) or [SBV2](https://github.com/litagin02/Style-Bert-VITS2) | 音声合成 (TTS) | ✅ |
| OBS Studio | 配信 | 配信時のみ |
| [わんコメ](https://onecomme.com/) | コメント取得 | コメント機能使う時 |
| LLM API キー | AI フリートーク・コメント返し | AI 機能使う時 |

### インストール手順

1. **VOICEVOX**（または SBV2）をインストールして起動
2. Chrome で `chrome://extensions` を開く
3. 「**デベロッパーモード**」を ON
4. 「**パッケージ化されていない拡張機能を読み込む**」→ このフォルダを選択
5. ツールバーの 📻 アイコンをクリック → ポップアップが開く

## 📖 基本的な使い方

### Step 1: ビューワーを起動

ポップアップの「🖥️ ビューワーを開く」をクリック。  
VRM キャラクターが表示されるタブが開きます。

### Step 2: コンテンツを読み込む

#### A. プロジェクトフォルダごと読み込む（推奨）

1. ポップアップの「📂 フォルダ」をクリック
2. プロジェクトフォルダを選択
3. フォルダ内の台本（`.md`）、朗読台本（`readings/`）、メディア（`media/`）を一括認識
4. 台本が複数ある場合は**プルダウン**で選択可能

#### B. 台本ファイルを直接読み込む

1. ポップアップの「📖 台本 (.md)」をクリック
2. セットリストまたは台本の `.md` ファイルを直接選択

#### C. Radio Station から再生

1. [Radio Station](https://k3peta.github.io/aituber-radio-station/) にアクセス
2. 拡張機能IDを接続（ポップアップの📋コピー → Station に貼り付け）
3. 番組カードの ▶ ボタンをクリック

### Step 3: 再生する

- **▶️ 再生** — 選択中の台本を再生
- **⏹️ 停止** — 再生停止
- **⏱️ 開始ウェイト** — 0〜60秒のカウントダウン後に再生開始（OBS切替用）

## 🖼️ PNGTuber

VRM モデルの代わりに、PNG 画像でキャラクターを表示する PNGTuber モードに対応しています。

### 使い方

ビューワーに **PNG 画像をドラッグ&ドロップ** するだけ。

| ドロップ | 結果 |
|---------|------|
| 画像 **2枚** | idle + talk で PNGTuber モード |
| 画像 **3枚** | idle + talk + blink（瞬き付き） |
| **Shift+ドロップ** 1枚 | PNGTuber モード（口パクなし） |
| 画像 1枚 | 背景変更 |
| **.vrm** ファイル | VRM モードに戻る |

### ファイル名の命名規則

自動判別するため、ファイル名にキーワードを含めてください。

| キーワード | 役割 |
|-----------|------|
| `talk`, `open`, `speak` | 口を開けた画像 |
| `blink`, `close` | 目を閉じた画像（瞬き） |
| 上記以外 | 通常（idle）画像 |

### クロマキー処理

グリーンバック画像は読み込み時に自動で透過処理されます。

### サンプル画像

`viewer/pngtuber/` に怪談ちゃんのサンプル画像（3枚）が同梱されています。

## 📹 録画機能

OBS なしで番組を動画として録画できます。

### 使い方

1. ビューワー右上の「📹 録画」ボタンをクリック
2. ボタンが赤くなり、タイマーが表示される
3. 番組を再生
4. **放送終了時に自動で録画停止 → WebM ファイルが自動ダウンロード**
5. 手動で停止したい場合はボタンを再度クリック

### 仕様

- **解像度**: 1920×1080
- **フレームレート**: 30fps
- **映像**: 背景 + キャラクター（VRM/PNGTuber）+ 字幕 + オーバーレイを合成
- **音声**: TTS・BGM・ジングルすべて含む
- **形式**: WebM（VP9+Opus）
- **ビットレート**: 4 Mbps

## 📻 セットリスト（台本）の書き方

> 詳細仕様は [SETLIST_FORMAT.md](https://github.com/k3peta/aituber-radio-station/blob/main/SETLIST_FORMAT.md) を参照

### ヘッダー（YAML frontmatter）

```markdown
---
title: 番組タイトル
speaker: 38          # VOICEVOX スピーカー ID（38 = ずんだもん ヒソヒソ）
speed: 0.90          # 読み上げ速度
loop: false          # ループ再生
pause_between: 1500  # セグメント間の間隔 (ms)
---
```

### セグメントタイプ一覧

| タイプ | 説明 | AI |
|--------|------|-----|
| `talk` | 台本トーク | 不要 |
| `jingle` | ジングル・SE 再生 | 不要 |
| `script` | 朗読 | 不要 |
| `freetalk` | AI フリートーク | 必要 |
| `reaction` | 朗読感想 | 必要 |
| `comments` | コメント読み上げ | 必要 |

### セグメント例

```markdown
# オープニング
[type: talk]
[bgm: media/bgm_chill.mp3]
[speaker: 3]
[emotion: happy]
[intensity: 0.7]
こんばんは！今日も楽しくやっていくよ！
```

### セグメント共通プロパティ

| プロパティ | 説明 |
|-----------|------|
| `speaker` | スピーカーID（セグメント単位で変更可） |
| `bgm` | BGM ファイル |
| `bgmVolume` | BGM 音量 (0.0〜1.0) |
| `bgmstop` | BGM 明示停止 |
| `emotion` | 表情（neutral/happy/sad/surprised/relaxed） |
| `intensity` | 表情の強さ (0.0〜1.0) |
| `overlay` | オーバーレイ画像 |

## 🔊 TTS エンジン設定

ポップアップの「🔊 TTS エンジン設定」から切り替えできます。

| エンジン | ポート | API |
|---------|--------|-----|
| **VOICEVOX** | 50021 | audio_query → synthesis |
| **Style-Bert-VITS2** | 5000 | GET /voice |
| **カスタム** | 任意 | VOICEVOX 互換 |

VOICEVOX 互換エンジン（COEIROINK、AivisSpeech 等）は「カスタム」でポートを変更するだけで使えます。

### 主なスピーカーID

| ID | キャラクター |
|----|------------|
| 2 | 四国めたん（ノーマル） |
| 3 | ずんだもん（ノーマル） |
| 8 | 春日部つむぎ |
| 29 | No.7（ノーマル） |
| 38 | ずんだもん（ヒソヒソ）← 通称「怪談ぺたん」 |

> 全スピーカー: VOICEVOX 起動後 `http://localhost:50021/speakers` で確認

## 💬 コメント連携（わんコメ）

わんコメが起動していれば HTTP API で自動取得。

- `[type: comments]` セグメントで読み上げ

- 「💬 ローカルチャット」でわんコメなしでも会話可能
